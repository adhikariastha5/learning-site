🧠 DeepCorpus — N-Gram Autocomplete

Using character n-grams and cosine similarity to predict/suggest words from a dictionary. Built in Clojure.

1. The Big Idea

Given a typed word (possibly misspelled), find the most similar words from a dictionary. The approach:

  • Break words into small character chunks (n-grams)
  • Represent each word as a count vector of those n-grams
  • Use cosine similarity to find the closest matches

2. What Are N-Grams?

An n-gram is a contiguous sequence of n characters from a word. This project uses 2-grams, 3-grams, and 4-grams.

5. Cosine Similarity

Cosine similarity measures how similar two vectors are, regardless of their magnitude. It returns a value between 0 (completely different) and 1 (identical direction).

similarity = (A · B) / (|A| × |B|)

Where:

  • A · B = dot product (multiply matching elements, then sum)
  • |A| = magnitude of vector A = √(sum of each element squared)
📝 Example: Similarity of "appp" vs dictionary words

vs "appleap" (A₁)

A₁ = [2, 1, 1, 1, 1, 0, 0, 0, 0]
B  = [1, 2, 0, 0, 0, 0, 0, 0, 0]

A₁ · B = (2×1) + (1×2) + 0 + 0 + ... = 4
|A₁| = √(4+1+1+1+1) = √8
|B|  = √(1+4) = √5

S₁ = 4 / (√8 × √5) = 4 / 6.32 ≈ 0.632

vs "ballap" (A₂)

A₂ = [1, 0, 0, 0, 0, 1, 1, 1, 1]
B  = [1, 2, 0, 0, 0, 0, 0, 0, 0]

A₂ · B = (1×1) + 0 + ... = 1
|A₂| = √(1+1+1+1+1) = √5
|B|  = √5

S₂ = 1 / (√5 × √5) = 1/5 = 0.200

Result: [0.632, 0.200] → highest similarity is at position 0 → predicted word is "appleap" ✅

6. The Algorithm (Step by Step)

Step 1: Load 1,000 random words from a 10,000-word dictionary
Step 2: Generate 2-grams, 3-grams, and 4-grams for every word
Step 3: Collect all unique n-grams across the dictionary → this becomes the "feature space"
Step 4: Build a count vector for each dictionary word (how often each n-gram appears)
Step 5: When the user types a word, build its count vector the same way
Step 6: Calculate cosine similarity between the input vector and every dictionary vector
Step 7: Return the top 5 most similar words

(defn singlegram [n, word]
  (for [i (range (+ (- (count word) n) 1))]
    (str/join (subvec (str/split word #"") i (+ i n)))))

(defn multigram [list_word] ...)

(defn word_vector [word]
  (let [gram (frequencies (multigram word))]
    (map #(get gram % 0) dictionary_unique_multigram)))

;; Cosine similarity between two vectors
(defn cos_similarity [v1, v2]
  (let [dot    (reduce + (map * v1 v2))
        mag1   (Math/sqrt (reduce + (map #(* % %) v1)))
        mag2   (Math/sqrt (reduce + (map #(* % %) v2)))]
    (if (<= (* mag1 mag2) 0) 0.0
      (float (/ dot (* mag1 mag2))))))

;; Find top 5 similar words
(defn similarity [word]
  (take 5 (sort-by first >
    (map vector
      (map (partial cos_similarity (word_vector word)) count_vector)
